Project Blueprint

MedFind Arifwala —
Complete Roadmap

90 din mein MVP launch. Solo builder. Zero external funding needed.

Phase 1 — Days 1-30
Research + Wireframes
Arifwala ki pharmacies count karo · 5 pharmacy owners se baat karo · Figma wireframes banao · Medicine categories list banao · Domain buy karo
Phase 2 — Days 31-60
MVP Build
Website + search functionality · Firebase database connect · Admin panel for pharmacies · 3-5 pharmacies manually onboard · WhatsApp bot basic version
Phase 3 — Days 61-90
Launch + Grow
10+ pharmacies onboard · Delivery network setup · Free month launch · WhatsApp groups mein promote · Feedback lo, improve karo · Phase 2 monetization start

💻 Tech Stack (Free/Cheap)

  • Frontend: HTML + CSS + JavaScript (aap jaante ho)
  • Database: Firebase Firestore (free tier)
  • Hosting: Vercel ya GitHub Pages (free)
  • WhatsApp Bot: Twilio API (free trial)
  • Admin Panel: Simple HTML form + Firebase
  • Domain: ~Rs.1500/year (medfind-arifwala.com)

💰 Revenue Model

  • Delivery charge: Rs.30 per order (Rs.70 driver ko)
  • Pharmacy listing: Rs.500-1000/month (Month 2 se)
  • Premium placement: Rs.1000/month
  • Target: 50 orders/day = Rs.1500/day = Rs.45,000/month
  • 10 pharmacies subscribed = Rs.5000-10,000/month extra
  • Break-even: Month 2-3

🎯 Target Users

  • Village areas from Arifwala — 20km radius
  • Elderly patients who can't travel easily
  • Emergency night-time medicine needs
  • Caregivers managing medicine for family
  • Pharmacies wanting more customers
  • District Health Authority (license tracking)

📱 Marketing Plan

  • Month 1: Free — word of mouth in Arifwala
  • WhatsApp groups: Share in all local groups
  • Pharmacy windows: Sticker lagao "Find us on MedFind"
  • Facebook: Arifwala local groups mein post
  • Village imams/community leaders — spread word
  • First 100 orders: Delivery free (acquire users)
